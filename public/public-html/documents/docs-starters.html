Getting started with {{startercont.productName}} starter App

By following this guide, you can use this service in your application

Prerequisites

  • NBOS account
  • Cloudfoundry
  • Eclipse IDE
  • GIT
1.Clone the Sample App

First, clone the sample App from Github repo

git clone https://gitlab.com/wavelabs/idn-api-spring.git
2.Run the App locally using the cmd

  • On cmd, change the directory to where the sample app is located

    cd idn-api-spring

  • Use Maven to install dependencies and buid the .war file

    mvn clean install

  • Run the App locally

    mvn spring-boot:run
Note: To stop the app-run press ctrl+c in the cmd
3.Prepare the app for deployment

To deploy to Bluemix, it can be helpful to set up a manifest.yml file. The manifest.yml includes basic information about your app, such as the name, how much memory to allocate for each instance and the route. We've provided a sample manifest.yml file in the get-started-java directory.
