Why this Service?

Wavelabs-OLX gives a platform to establish communication between Seller and Buyer. It’s very hard to letting everyone know you have product to sell And for buyers also it’s very difficult to them directly searching for already used products for sell. To avoid this problem our wavelabs-olx providing platform.

Features

    Events on Advertisement

  • Posting
  • Update
  • Delete
  • Read

    Events on Comment

  • Posting
  • Update
  • Delete
  • Read

Services

To hit the Service

End Point: https://nbos_tech_talks.cfapps.io/advertisement/1

Set the header with accesstoken

Syntax: access_token : {token_values}

Post Your Add?

Observe, below Image to post advertisement with attributes title, category and Ad Description. In addition to that description you also need to pass the User object with attributes id.

Click to see the image

End Point: https://nbos_tech_talks.cfapps.io/advertisement

"description": "Duke bike for 70k In kondapur, contact : 9032118864",
